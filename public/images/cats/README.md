# Cat images

Replace the `placeholder-*.svg` files in this folder with your own photos.

Recommended:

- Use **WebP** or high-quality **JPEG** for smaller file sizes.
- Keep a consistent aspect ratio (for example **4:3**) for gallery harmony.
- Update paths in `src/content/cats/{slug}.json` (`coverImage` and `gallery`) to point to your new files under `/images/cats/`.

After swapping images, run `npm run build` to verify everything compiles.
